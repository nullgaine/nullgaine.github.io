概念核研究計画（通称：Cプロジェクト）

内部技術記録 抜粋（1992年時点）

本計画は、従来の遺伝子・環境要因では説明不可能な細胞挙動の同期現象を起点として開始された。
当初は測定誤差または外的要因と判断されたが、複数環境下での再現性が確認されたことにより、未知の情報構造の存在が仮定された。
本資料は、当該構造の特定および操作手法確立までの経緯をまとめたものである。

1. 初期観測（1988–1989）

1.1 観測対象

  • 人工授精により得られた受精卵由来細胞群
  • 同一環境下で培養された複数サンプル

1.2 異常現象

  • 異なる細胞群間での同時的な反応変化
  • 外部刺激に対する過剰な同期応答
  • 遺伝的差異を無視した反応パターンの一致

当該現象は、遺伝子情報および既知の細胞間通信機構では説明できない。

2. 再現性確認（1990）

2.1 手法

  • 別施設・別機器による再試験
  • 刺激条件の完全固定
  • サンプル数の増加

2.2 結果

同一刺激条件下において、異なる個体由来細胞群が同一の応答系列を示すことを確認。

2.3 結論

本現象は偶発的事象ではなく、再現性を持つ現象であると判断された。

3. 挙動解析（1990–1991）

3.1 持続性試験

外部刺激を停止した後も、細胞群は一定の応答パターンを維持した。
→ 一時的反応ではなく、内部構造としての固定化が示唆される

3.2 伝播性試験

処理済み細胞群と未処理細胞群を共培養した場合、未処理群にも同一の応答パターンが出現。
→ 情報が物理媒体を介して移動している可能性

3.3 自己再現性

分裂後の細胞においても、同一パターンが維持された。
→ 情報構造は複製される

4. 概念核の仮定（1991–1992）

4.1 問題提起

  • 保持される
  • 伝播する
  • 再現される

これは従来の生体情報の枠組みを超える。

4.2 定義

概念核（Concept Core） と定義する。

4.3 特性

  • 刺激が消失しても維持される
  • 他細胞へ伝播する
  • 同一構造へ収束する

4.4 仮説

概念核は、生体情報とは別系統の「非物理的情報構造」である可能性。

5. 抽出試験（1992）

5.1 手法

  • 応答維持率を指標とした細胞群選別
  • 刺激除去後の安定性を比較

5.2 結果

特定の細胞群において、長期間にわたり応答パターンが維持されることを確認。

5.3 所見

概念核は完全な物理分離は不可能であるが、
機能的に抽出可能と判断される。

6. 再構築理論（1992）

6.1 仮説

未分化細胞に概念核を導入することで、発生過程において当該構造が固定される可能性。

6.2 手法概念

  • 概念核保持細胞群の選別
  • 未分化細胞との接触・共培養
  • 初期発生段階での固定化

6.3 目的

概念構造を内包した個体の生成。

7. 人体適用に関する判断（1992）

概念核は再現性・伝播性・発生影響を持つことから、
生体個体への適用が可能と推定された。

7.1 倫理審査

正式な倫理審査は行われていない。

7.2 実験移行

1993年より、生体生成実験へ移行。

■補足記録

1989年・1990年出生個体

  • 概念核導入を伴わない人工授精個体
  • 概念構造の安定保持は確認されていない

公式記録：廃棄済
内部記録：保留
外部共有：なし

備考

本研究における「概念」は宗教的・哲学的概念とは異なる。
生体内において保持・伝播・再現される情報構造を指す。

概念核の安定性および個体への影響は未確定であり、
人体適用には重大なリスクが伴う。
